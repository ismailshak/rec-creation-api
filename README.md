# Rec|Creation API

This API was created to support the [Rec|Creation](http://rec-creation.surge.sh/) application.

Data for games was collected to initially seed the database.

[Deployed API](https://rec-creation-api.herokuapp.com/api/games)

[API Documentation](https://rec-creation-api.herokuapp.com)

## Frameworks Used

Express, Mongoose, MongoDB, apidocs
